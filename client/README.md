# developer setup
## prerequisites
- [Node.js](https://nodejs.org/en/)

## setup
1. Clone the repository
2. `cd client`
3. `npm install`
4. `npm start`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## build
1. `npm run build`
